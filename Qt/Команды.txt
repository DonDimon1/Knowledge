QWidget.sizeHint() - возвращает некоторое рекомендованное значение размера виджета 
QWidget.minimumSizeHint() - минимальное значение виджета

sizePolicy - стратегия изменения размера. Определяет то, как объект компоновщика будет интерпретировать рекомендованный размер.
	Horizontal Policy: политика по горизонтали
	Vertical Policy: политика по вертикали
	Horizontal Stretch:
	Vertical Stretch:
	
	Fixed: используется только рекомендованный размер
	Minimum: Используется минимальный размер. Виджет не может быть меньше чем минимальный размер.
	Maximum: Виджет не может быть больше чем максимальный размер.
	Preferred: Виджет стремиться иметь рекомендованный размер.
	Expanding: Виджет стремиться расшириться.
	MinimumExpanding: Стремиться расшириться, при этом виджет не может быть меньше чем минимальный размер.
	Ignored: Игнорирование значение метода sizeHint(). Виджет может полностью схлопаться. 


width: parent.width - ширина как у родительского элемента.
