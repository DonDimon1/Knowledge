//argc argv 
//Они нужны для передачи в программу параметров
argc - arguments count(кол-во аргументов) .Кол-во не может быть меньше 1.
argv - arguments values(значения аргументов). Первый параметр всегда путь к exe файлу нашей программы.

Например в консоли винды мы можем запустить такую команду: ping google.com
С помощью неё мы узнаем нашу задержку до сайта гугла. Так во мы запустили программу ping и положили в неё параметр google.com. 
Мы можем описать логику программы так, что для разных параметров поведение программы может изменяться.

Если запустить нашу программу через консоль винды и задать ей несколько параметров.


void main(int argc, char* argv[])
{
	cout << argv[i] << endl;// Наша программа просто выводит все параметры, которые мы в неё вложили при запуске.
}

Если мы не хотим чтобы в нашей программе была реализация доп. параметров, то просто не указываем их в main:
void main()
{
	
}

int main или void main
Если программа отработала корректно, то main возвращает 0. Если нет, то возвращается другое знаение(например код ошибки).
int main(int argc, char* argv[])
{
	return 0;
}
