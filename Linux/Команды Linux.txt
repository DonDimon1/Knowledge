Команды Linux:

cd - сменить дириктория
cd .. - поднятся наверх
pwd - узнать в какой дириктории мы сейчас находимся                                                       
cd - - возвращает туда где мы были до этого
cd без параметров - вернуться домой
ls - отобразить список файлов в текущей дириктории
ls -l - отобрази подробно про каждый файл
cat - вывести содержимое файла на экран
rm - удалить файл

пример файла:
lrwxrwxrwx

типы файлов:
- обычный файл
l - ссылка
d - дириктория

Права файлов:
r - право на чтение
w - право на запись
x - прово на выполнение
 
Права доступа к файлу бывают 3ёх категорий:
lrwxrwxrwx
l rwx (- права доступа к этому файлу у валадельца этого файла) rwx (- права к этому файлу у группы владельцев этого файла) rwx (- права у всех остальных)


Дальше Команды:
Ctrl + w - стереть слово
Ctrl + u - стереть строку

chmod - менять права chmod -x (отобрать права на выполние) chmod +x (дать права на выполние)
